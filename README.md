# Live Sports Highlight Generator

## Overview
A web-based application that automatically detects and extracts key moments from sports videos using audio and visual analysis.

## Features
- Crowd cheer detection
- Scoreboard change detection
- Automated highlight generation
- Video upload support
- React dashboard
- Flask backend processing

## Technologies Used
- Python
- Flask
- React
- OpenCV
- Librosa
- FFmpeg

## Project Structure
- detector/ : Highlight detection modules
- frontend-ui/ : React frontend
- main.py : Processing logic
- webserver.py : Backend server

## Future Enhancements
- Live match processing
- AI-based event detection
- Multi-sport support
